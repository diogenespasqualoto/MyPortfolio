Conhecimentos

HTML5

O HTML é a base de todo sistema web, é com ele que adicionamos em nossa página os Textos, Imagens, Formulários, Vídeos e etc.

CSS

Com o CSS adicionamos estilos em nossa página: Cores, Tamanhos, Posicionamentos Sombras, Grids, Efeitos, Animações e etc.

Design Thinking

Um método de desenvolvimento de produtos e serviços que enfoca as necessidades, desejos e restrições dos usuários. O principal objetivo do design thinking é traduzir dificuldades e limitações nos interesses dos clientes e no valor comercial da empresa

JavaScript

Com o Javascript as possibilidades são enormes, validando formulários, processando dados, manipulando elementos na tela, consultando APIs, etc.

bootstrap

O framework bootstrap desenvolvido pelo Twitter visa padronizar as ferramentas da empresa. A estrutura combina CSS e JavaScript para estilizar os elementos da página HTML. Ele permite mais do que apenas alterar a cor de botões e links.

Git e Gituhub

Git é um sistema de controle de versão de projeto. Com ele, vamos registrar o histórico de edições no arquivo de sistema.
GitHub é um código-fonte de controle de versão e plataforma de hospedagem de arquivos usando Git. Você aprenderá como armazenar e criar uma versão de seus projetos pessoais e as etapas necessárias para criar ou contribuir com um projeto de código aberto no GitHub.
